# Taskwondo Overview

A visual walkthrough of Taskwondo's features, from everyday essentials to advanced configuration.

## Project Overview

The project overview gives you a quick snapshot of your project: open items by type, recent activity trends, and project description.

![Project Overview](screenshots/taskwondo-project-overview.png)

## Work Items

List all work items with filters for milestone, type, priority, status, and assignee. Sortable columns and cursor-based pagination keep large backlogs manageable.

![Work Items](screenshots/taskwondo-workitems.png)

## Inbox

Your personal triage queue. Drag items to reorder priorities, toggle auto-hide for completed items, and quickly create new work from here.

![Inbox](screenshots/taskwondo-inbox.png)

## Watchlist

Track work items across all projects in one place. Switch between list and board views and filter by type, priority, status, or assignee.

![Watchlist](screenshots/taskwondo-watchlist.png)

## Milestones

Group work items into milestones with due dates and progress bars. See at a glance how many items are open, closed, and what percentage is complete.

![Milestones](screenshots/taskwondo-milestones.png)

## Milestone Dashboard

Dive into a single milestone for detailed stats: completion percentage, item breakdowns by type, priority, and label, plus time tracking summaries.

![Milestone Dashboard](screenshots/taskwondo-milestone-dashboard.png)

## Workflows

Define custom workflows with statuses, transitions, and categories. System workflows are included out of the box, and you can create project-specific ones.

![Workflows](screenshots/taskwondo-workflows.png)

## Workflow Editor

Add or remove statuses, set their categories, reorder them, and define allowed transitions between statuses.

![Workflow Editor](screenshots/taskwondo-workflows-edit.png)

## Workflow Mapping & SLA

Assign workflows per work item type and configure SLA business hours with timezone support.

![Workflow Mapping & SLA](screenshots/taskwondo-workflows-mapping.png)

## User Preferences — Appearance

Choose your theme (light, dark, or system), font size, and language. Nine languages are supported, including RTL Arabic.

![User Preferences — Appearance](screenshots/taskwondo-userpreferences-appearance.png)

## User Preferences — API Keys

Create and manage API keys for programmatic access. Set permissions, expiration, and revoke keys at any time.

![User Preferences — API Keys](screenshots/taskwondo-userpreferences-apikeys.png)

## System Settings — General

Configure instance-wide settings like the brand name displayed across the UI.

![System Settings — General](screenshots/taskwondo-systemsettings-general.png)

## System Settings — Authentication

Enable or disable login methods, toggle email registration, and configure OAuth providers with drag-to-reorder.

![System Settings — Authentication](screenshots/taskwondo-systemsettings-authentication.png)
